Module: tests/populatedAndOfTypeMocha

populatedAndOfType: checks if a data layer property is populated and of type (according to utag.ut.typeOf logic: "regexp, string, object, number, boolean"). If property is an Array, all Array members are checked.

Special commands:

 - preceding "**", e.g. "**string" -> optionally populated, but if populated, should be of type
 - "!!" -> must not be populated -> throws error if populated
 - If property is an object, it will run a logical test, e.g. a switch (similar to fullOrRegExMatch) (see TMSHelper.logicalTest)
Parameters:
Name Type Description
prop string

data layer property (key) to check whether populated and type
type string | object

to check - if string = expected type (e.g. "string"), if object = logical test
dl object

full data layer
